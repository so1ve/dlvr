GitHub

/gh/[USER]/[REPO]@[BRANCH]/[PATH]

可选参数

  • min  /  minify ：是否压缩（目前支持JS、JSON、CSS） 例如： 
    /npm/nprogress/nprogress.css?min
/npm/nprogress/nprogress.css?minify
    出于稳定性考虑，我们不使用JSDelivr风格的.min.ext压缩方式。倘若作者在日后加上了对应的min文件，这将有可能导致原本的文件（经过DLVR压缩后）和作者添加的文件内容不匹配。
